1. Project Description

Enroute is intended to help users plan itineraries for various trips. Users can collaborate on trips with other users, making Enroute an interactive and fun platform to plan vacations.

2. Features

  • Event Planner
  • Shared Gallery
  • Group Chat
  • Interactive Map
  • Location Tracking
  • Cost Breakdown Assistant

3. Installation

To download Enroute, go to the App store on the iPhone. Search for ‘Enroute’ in the search box and select ‘GET’. Enroute 1.0 is free to download from the App store. No payment is required. Select on the application once EnRoute is downloaded on your mobile device.

4. Components & Technologies

  • Google Maps
  • Yelp API
  • db4free
  • Xcode
